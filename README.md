R-Code and data for a paper titled "Preference Versus Actual Choice: Evidence From a Payments Survey"

by Oz Shy

---------------------------------------

Instructions:

Download the R-code file: "actual_2026_mm_dd.R" and the two data files: "dcpc-2024-indlevel-public.RDS" and "dcpc-2024-tranlevel-public.RDS" into the same folder on your computer.

Run the ENTIRE code first (don't forget to change the working directory).

If you are using RStudio, after you run the ENTIRE code first, you can jump to the beginning and end of each table and figure by clicking the list of contents on the left-lower corner. 

However, for the yearly 2021-2025 analysis (preparation for the graphs/tab;es) there are repeating variables that then get saved by year.
